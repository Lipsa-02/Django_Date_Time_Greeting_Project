# Django_Date_Time_Greeting_Project
A simple Django web application that greets users based on the current time of the day.  This project demonstrates how to use Python's datetime module with Django views and templates to dynamically display greetings such as Good Morning, Good Afternoon, or Good Evening when a user visits the webpage.
🚀 Features
Detects the current system time
Displays greeting messages dynamically
Uses Django views, templates, and URL routing
Beginner-friendly Django project
Simple logic using Python's datetime module

🛠️ Technologies Used
Python
Django
Datetime module

⚙️ How It Works
The Django view fetches the current system time using Python's datetime.
Based on the hour of the day, the application decides the greeting:
Morning: Good Morning
Afternoon: Good Afternoon
Evening: Good Evening
The greeting message is passed to the HTML template and displayed on the webpage.
